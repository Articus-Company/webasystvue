## Установка

```bash
pnpm add webasystvue
```
```bash
npm install webasystvue
```
```bash
yarn add webasystvue
```

## Использование

```vue
<template>
    <!-- Используйте компонент -->
    <Badge>
        content
    </Badge>
</template>

<script setup lang="ts">
    import { Badge } from 'webasystvue' // Импортируйте компонент
</script>
```

## Разработка

### App (docs)
```bash
pnpm i
cd app
pnpm dev
```

### Package
```bash
pnpm i
cd packages/core
pnpm watch
```

### Histoire
```bash
pnpm i
cd .histoire
pnpm story
```
