<h1 align="center">
WbasystVue
</h1>

<p align="center">
Задекларированная <a href="https://vuejs.org/">Vue 3</a> библиотека компонентов <a href="https://www.webasyst.ru/">Webasyst</a>.
<p>

## Установка

```bash
pnpm add webasystvue
```
```bash
npm install webasystvue
```
```bash
yarn add webasystvue
```

## Разработка

### App (docs)
```bash
pnpm i
cd app
pnpm dev
```

### Package
```bash
pnpm i
cd app
pnpm dev
```

### Histoire
```bash
pnpm i
cd .histoire
pnpm story
```
